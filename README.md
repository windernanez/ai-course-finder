# AI Course Finder 🚀

Encuentra los mejores cursos de programación en YouTube usando Inteligencia Artificial.

## Características
- 🤖 Búsqueda inteligente con IA
- 📊 Clasificación automática por nivel (principiante/intermedio/avanzado)
- 🎯 Recomendaciones personalizadas
- 📈 Estadísticas de aprendizaje
- 🌐 Interfaz moderna y responsive

## Tecnologías
- **Backend**: Python, Flask, scikit-learn, YouTube API
- **Frontend**: HTML5, CSS3, JavaScript
- **IA**: Naive Bayes para clasificación de niveles

## Despliegue en Render
1. Conecta este repositorio a Render
2. Configura las variables de entorno
3. ¡Listo!
